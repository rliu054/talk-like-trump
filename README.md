# Tweet like Trump
Train an RNN model to generate Trump style tweets.

## Dependencies
* Python 3.5+
* PyTorch 0.4.0

## Usage
### 1. Clone the repo
```bash
$ git clone https://github.com/rliu054/tweet-like-trump
$ cd tweet-like-trump/
```
### 2. Run the model
```bash
$ python main.py --embed_size=10 --hidden_size=10 --epochs=10
```
